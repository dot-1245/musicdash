# 🎵 MusicDash Pro

アルバムアートの色に合わせてUI全体が美しく染まる、スタイリッシュな音楽再生ダッシュボードです。  
デスクトップでの閲覧はもちろん、スマホを横向きに固定して「専用ミュージックモニター」として使うことに特化しています。

## ✨ 主な機能

- **🎨 動的アクセントカラー**: 再生中のアルバムアートから最適な色を自動抽出し、UIの配色をリアルタイムに変更。
- **📱 スマホ最適化**: 画面のどこかをタップするだけで横画面（Landscape）にロック。
- **🎼 マルチプレイヤー対応**: `playerctl` を介して、Spotify、VLC、ブラウザなどのプレイヤーを自在にコントロール。
- **🌙 3つのテーマ**: LIGHT / DARK / AMOLED のモード切り替えが可能。
- **🕰️ ユーティリティ**: 時計、日付、バッテリー残量の表示機能を搭載。
- **✂️ スマートタイトル表示**: 長い曲名やアーティスト名を適切に短縮（25文字制限）し、レイアウト崩れを防止。
